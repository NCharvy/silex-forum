1. Tout d'abord, copier le fichier forumphp.conf dans /etc/apache2/sites-available
2. Pour activer le Virtual Host, utiliser la commande : sudo a2ensite forumphp.conf
3. Ajouter dans /etc/hosts : 127.0.0.1		domaineduforum (j'ai mis local.forumphp.com)
4. Redémarrer le PC pour prendre en compte les modifications du fichier hosts
5. Mettre à jour le .htaccess, il suffit de modifier le RewriteBase et mettant uniquement /
6. Redémarrer le service apache : sudo service apache2 restart
